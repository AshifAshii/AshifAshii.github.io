Time Capsule

A personal diary application that is used to add, update, and viewing registered users' memories along with their information which will be private only to the user. The data are stored in the remote MySQL database. Desktop, web, and mobile applications were developed in which the data is synchronized across all. The desktop application was developed using Java and its Swing API. The web application was developed using Java Servlets, JSP, and JavaScript. The mobile application was developed using Android XML and Java with PHP API and is connected to the client through the Android Volley library. Push notification is enabled through the firebase platform by manipulating FCM tokens. The application is deployed in production using the Heroku platform.

  • APK: Time Capsule
  • Date: Jul 2019 – May 2020
  • Service: Android Application

Mess Forum

A portal application developed for maintaining students' data, hostel food count, token registrations, and notifications. It includes both Student and Admin modules with a cloud database using MySQL in which data is synchronized across all platforms. The web application was developed using HTML, CSS with PHP as a backend. The mobile application was developed using Android XML and Java with Node.js API as backend connected using the Retrofit library of Android. The push notification feature is achieved by integrating the Google firebase with the application through manipulating FCM(Firebase Cloud Messaging) tokens. The application is deployed in production using the Heroku platform.

  • Admin APK: Mess Forum - Admin
  • Student APK: Mess Forum - Student
  • Date: Aug 2019 – Nov 2019
  • Service: Android Application
